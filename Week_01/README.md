## 职业规划及发展

+ **You** are the owner of your career.

+ 职业发展三要素（互相促进）
  + 晋升
  + 成就
    + 业务型
      + 业务目标
        + 理解业务目标并转化为可量化指标
      + 技术方案
        + 形成纸面方案、完成小规模试验
      + 实施方案
        + 确定实施目标与人员
        + 管理进度
      + 结果评估
        + 数据采集汇报
    + 工程型
      + 目标
        + 质量与效率
      + 方案与实施
        + 规章制度
        + 库
        + 工具
        + 系统
      + 结果评估
        + 线上监控
    + 技术难题
      + 目标
        + 公认技术难点
      + 方案与实施
        + 依靠编程力及架构能力形成解决方案
      + 结果评估
        + 解决问题
  + 成长
